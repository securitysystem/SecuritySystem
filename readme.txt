This is meant to run a security system on Raspberry Pi running Debian/Raspbian.

1. Copy 'home-pi-securitysystem' to /home/pi/securitysystem
2. Install apache2
3. Install PHP5
4. Set file permissions for 'www-data' and 'pi' by putting them in the same user group
5. Copy 'varwwwfolder' to /var/www
6. Run the Security System.
